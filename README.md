# This repository implements the complete CARPOS system. 

The main objective of the CARPOS integrated system is to demonstrate the functionality, robustness, and adaptability of the integrated robotic framework, to evaluate its performance against the predefined Key Performance Indicators (KPIs), and to highlight its contribution to coachable, automated, flexible, and sustainable harvesting. In addition to presenting performance results.
